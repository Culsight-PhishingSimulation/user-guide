User Management

Culsight - Phishing Simulation supports user accounts with different roles. Right now, we offer two separate roles that can be assigned to users:

  • User - This role allows the user to do anything except system-level administrative tasks, such as managing users, managing webhooks, etc.
  • Admin - This is a system-level administrative role that has full permissions to manage the Culsight - Phishing Simulation installation

Delete a User

To delete a user, click the red trash can icon next to the username in the users list.
