Tools

****GoReport
****Author: Chris Maddalena
A Python script to collect campaign data from Culsight - Phishing Simulation and generate a report

****Phishbuckets
****Author: snori74
Command-line scripts to manage phishing campaigns with API calls to a Culsight - Phishing Simulation server

****Lure
****Author: Jayme (highmeh)
Lure assists in phishing target collection by pulling and parsing email addresses for a target organization. The results are normalized into a format recognized by Culsight - Phishing Simulation, and then uploaded to the server.

****GoLDAP
****Author: Mohammed AlHowsa
A Python script that imports users from active directory to Culsight - Phishing Simulation.

****Phish Paste
****Author: Glenn Wilkinson
Phish Paste is a tiny utility to copy templates, pages, and sending profiles between different Culsight - Phishing Simulation user accounts.

****DW Culsight - Phishing Simulation Reporter
****Author: Daniel Wellington Technologies
This plugin makes it easy to report phishing for endusers. It will report using the native report function in Culsight - Phishing Simulation if it is a Culsight - Phishing Simulation phishing mail, if not it will capture all headers and content and send it to a mail defined where a security team can take a closer look

****DW Culsight - Phishing Simulation Automation
****Author: Daniel Wellington Technologies
This project is intended to automate the procedure internal phishing campaigns via Culsight - Phishing Simulation as well as avoiding the Culsight - Phishing Simulation URL being flagged by google safebrowsing. The idea is to run it completely by itself by scheduling the scripts to run in the prefered manner.
